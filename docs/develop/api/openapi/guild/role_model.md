# 频道身份组对象(Role)

### Role
| 字段名        | 类型                                                   | 描述                            |
| ------------ | ----------------------------------------------------- | ------------------------------ |
| id           | string                                                | 身份组ID                        |
| name         | string                                                | 名称                            |
| color        | uint32                                                | ARGB的HEX十六进制颜色值转换后的十进制数值 |
| hoist        | uint32                                                | 是否在成员列表中单独展示: 0-否, 1-是 | 
| number       | uint32                                                | 人数                            |
| member_limit | uint32                                                | 成员上限                         |

### DefaultRoleIDs(系统默认生成下列身份组ID)
| 身份组ID默认值 | 描述 |
| --- | --- |
| 1 | 全体成员 |
| 2 | 管理员 |
| 4 | 群主/创建者 |
| 5 | 子频道管理员 |
